# grunt-cloudock

### Overview
A Grunt plugin to manage dockers on clusters using dockerode and pkgcloud.

### Usage:
##### Manage cluster VMs:
grunt clouddock:node: [ create | list | destroy ]
##### Manage cluster security groups:
grunt cloudock:secgroup:[ create | list | update | destroy ]
